Nueva funcionalidad en Tractis: “Variables” en plantillas y contratos

Hoy lanzamos una nueva funcionalidad llamada “variables” que te hará ser aún más productivo al trabajar con plantillas y contratos.

La tortura de personalizar contratos

Tradicionalmente, las plantillas suelen incluir las típicas líneas de puntos para indicar a las partes donde deben personalizar el contenido del contrato. El problema es que, incluso con la línea de puntos, a veces es difícil para un no-abogado saber qué hay que poner en cada sitio. Las mejores plantillas son aquellas que, en vez de líneas de puntos, incluyen notas o “variables” como “Nombre del comprador“, “Dirección de envío“, etc.

Aún así, cuando la misma variable (p.ej: “Nombre del comprador“) se repite 40 veces a lo largo de un documento de 40 páginas es muy pesado tener que rellenar esa variable una y otra vez. ¿No sería genial que Tractis “leyese” el documento, te preguntase en un formulario por todas las variables encontradas y después las rellenase por tí sin importar cuantas veces se repitan?. Pues eso es precisamente lo que hemos hecho.

Creando una variable en Tractis

Para crear una variable en una plantilla o contrato, basta con que escribas el nombre de la variable entre corchetes prestando atención a no dejar espacios (p.ej: [CALLE], [DNI], [CIF] o [PVP]).

Rellenando una variable en Tractis

Para rellenar las variables del contrato, pulsa en “Variables” debajo del título del documento. Accederás a un formulario donde tendrás que rellenar cada variable “una sola vez”.

… y, al guardar, Tractis reemplazará en masa todas las ocurrencias de cada una de las variables en el contrato.

Si ves que te has equivocado en algún dato, puedes volver a Variables, corregirlo y volver a guardar. Una sola vez y en un único sitio.

Dominando la técnica: Agrupando variables

Puedes agrupar variables incluyendo barras para indicar cada nivel de agrupación. P.ej: [EMPRESA1/CIF], [EMPRESA1/CIUDAD], [EMPRESA1/ESCRITURA/FECHA_DE_OTORGAMIENTO], agrupará los datos de la siguiente forma:

Pulsa en “Crear un contrato con esta plantilla” y después visita el formulario de variables. Verás lo rápido que podrás adaptar el contrato a tu caso concreto.

Ventajas de trabajar con variables

Trabajar con variables tiene muchas ventajas:

  1. Ahorro de tiempo: Si utilizas variables conseguirás enormes ahorros de tiempo cuando personalices un contrato o el borrador inicial de una negociación.
  2. Sencillez: Sí, Word y otros procesadores de texto ofrecen una funcionalidad similar pero, reconozcámoslo, es tan complicada que casi nadie la utiliza. Al final, acabas recurriendo a “Buscar y reemplazar” para cada variable. Tractis ofrece justo lo que necesitas y nada más.
  3. Flexibilidad: Este sistema de variables no impone una especificación única y cerrada de variables que todos los usuarios deban utilizar sino que permite que cada usuario o empresa cree su propia especificación de variables y la nombre como prefiera.
  4. Potencia: Planeamos lanzar la API de Tractis en Noviembre. Será todo un hito en el desarrollo de Tractis y dará una nueva dimensión a la plataforma. Entre otras cosas, la API permitirá trabajar con “variables” desde el principio. Las empresas podrán generar miles de contratos, personalizados para cada uno de sus clientes, y enviarlos de forma automatizada.

Anímate a utilizar variables en tus plantillas y contratos. Solo tienes que recordar encerrar tus variables entre corchetes y utilizar barras si quiere agruparlas. Fácil, sencillo y potente.
