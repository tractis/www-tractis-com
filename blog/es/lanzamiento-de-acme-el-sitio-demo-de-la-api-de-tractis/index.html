Lanzamiento de ACME: El sitio demo de la API de Tractis

La semana pasada os dábamos un adelanto de la API en la que estamos trabajando y comentábamos algunos de los servicios que iba a prestar:

  • Autenticación de usuarios con certificados electrónicos.
  • Automatización del envío de contratos personalizados a clientes.

Hoy queremos hablaros de algunas de las cosas que ya tenemos listas. Por más que elaboremos documentación, lo mejor para entender las posibilidades de la API es verla en acción. Hemos decidido crear una empresa ficticia llamada “ACME”, cuyo sitio sitio web (http://acme.tractis.com) recogerá una lista de demostraciones de todo lo que podeis hacer con la API de Tractis.

ACME: Demo de envío de contratos personalizados a clientes

La primera demostración que recoge el sitio web de ACME es sobre el “Sistema de gestión de contratos”. Este sistema te permite utilizar la API de Tractis para automatizar la creación y envío de contratos totalmente personalizados a tus clientes. Tal como verás en la demo, con la API podrás:

  • Personalizar el asunto y el cuerpo del e-mail que recibirá el cliente.
  • Personalizar las instrucciones que aparecerán encima del contrato. Esto es útil para explicar al cliente los pasos a seguir (“revise el contrato y, si está de acuerdo, pulse el botón de “Firmar” al final del mismo”), solicitar el envío de información adicional, recordar plazos de entrega, etc.
  • Seleccionar la plantilla que servirá de base al contrato. Puedes elegir una de las plantillas de ejemplo o iniciar sesión y utilizar las plantillas de tu Librería privada.
  • Rellenar las variables de la plantilla que vas a utilizar. Lógicamente, esta opción solo está disponible si la plantilla utilizada incluye variables. En esta demo, todos los contratos envíados comparten las mismas variables (P.ej: Nombre del comprador = “Juan García”). Sin embargo, con la API podrás personalizar las variables para cada contrato.

Algunos detalles para “los comerciales”

Tus clientes son tuyos:

  1. No tienen porque registrarse en Tractis para firmar el contrato.
  2. Recibirán un e-mail personalizado con el remitente, asunto y cuerpo del mensaje que tu decidas.
  3. Accederán a un contrato personalizado con la imagen corporativa de tu compañía (logo, colores, instrucciones).

Algunos detalles para “los técnicos”

Todavía estamos confeccionando la documentación de la API. Sin embargo, os podemos adelantar algunas cosas:

  • Acceso: inicialmente la API será accesible vía REST y, más adelante, incluiremos también la posibilidad de acceso vía SOAP.
  • Formato: Utilizamos XML como formato de representación. En la demo actual podéis ver la previsualización del XML de las peticiones a Tractis. En breve, incluiremos la posibilidad de ver la traza de peticiones a la API.

El trabajo con la API continúa… ¿Tú qué necesitas?

En Tractis creemos firmemente que el potencial de internet para los negocios va más allá de la promoción y la comunicación. Ahora también puedes negociar y contratar de forma segura 100% online y con las garantías legales necesarias. Ya dispones de la tecnología y la legislación.

Si eres de los que creen que su empresa podría aprovecharse de este tipo de servicios, no lo dudes, ponte en contacto con nosotros. Estaremos encantados de escuchar tus ideas para agilizar e incrementar tus negocios con internet y la firma electrónica.

Por Ernesto Jiménez
