¿Cómo me apunto al Programa de Afiliados de Tractis?

Para apuntarte al Programa de Afiliado de Tractis, sigue estos pasos:

Paso 1: Crea, completa y firma tu “Contrato de Afiliado referenciante de Tractis”

Este documento describe el acuerdo genérico entre tú, como Afiliado, y Tractis.

Para crear el contrato, pulsa este botón y sigue las instrucciones:   

Paso 2: Crea, completa y firma tu “Solicitud de cliente potencial a Tractis”

Este documento describe el acuerdo específico entre tú y Tractis para un cliente potencial concreto. Deberás rellenar y firmar una de estas solicitudes por cada oportunidad o cliente potencial que detectes. Revisaremos cada solicitud y, si decide aceptarla, la firmaremos para que tengas tranquilidad de que esa cuenta es tuya y solo tuya.

Para crear una solicitud, pulsa este botón y sigue las instrucciones:   

Paso 3: Manténte al tanto de las novedades en el Programa de Afiliados de Tractis

Asegúrate de marcar la casilla “Afiliado” en tu perfil en Tractis. Así estarás al tanto de las todas las novedades que se produzcan en el Programa de Afiliados.
