¿Qué nivel de garantía ofrece una “autenticación con OpenID”?

La autenticación con OpenID ofrece un nivel de garantía mínimo (Tractis Score 1).

Nivel de garantía asociado a una “autenticación con OpenID”
Tractis Score Nivel de garantía Ejemplos
TS1 Mínimo
  • Usuario autenticado utilizando usuario y contraseña como login.
  • Usuario autenticado con una cuenta Open ID enlazada a una cuenta que utiliza usuario y contraseña como login.

Nota importante.
No todos los métodos de autenticación ofrecen los mismos niveles de garantía. En función del nivel de seguridad que requieras, deberás utilizar un método de autenticación u otro. Para ayudarte a decidir el método de autenticación ideal para tu caso, hemos creado una clasificación de niveles de garantías, o “Tractis Scores“, que van desde “Tractis Score 1 – Nivel de garantía mínimo” a “Tractis Score 4 – Nivel de garantía máximo“. Si deseas conocer más información sobre Tractis Score: 

  • El documento “Tractis Score Policy” describe los criterios utilizados para determinar el Tractis Score de un método de autenticación.
  • El documento “Tractis Score Mapping” lista completa de métodos de autenticación posibles y sus correspondientes Tractis Score.
