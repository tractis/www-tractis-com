¿Qué significa el color del borde de mis contratos?. ¿Por qué cambia?

El color del borde de tus contratos te indica rápidamente si el estado del documento (abierto, bloqueado, firmado, etc.) . Este color cambia a medida que el contrato cambia de estado.

Tractis utiliza los siguientes códigos de colores:

  • Verde: El documento está “abierto” y tienes permisos para editarlo (eres “Administrador” o “Editor”).
  • Rojo: No tienes permisos para editar el contrato, eres “Lector”.
  • Violeta: Estás viendo una “comparación de versiones” del documento. Nadie puede editar una comparación, solo la última versión es editable.
  • Naranja: Has “bloqueado” el documento. Nadie, aparte de ti, puede editar el contrato hasta que lo desbloquees.
  • Amarillo: El documento ha sido bloqueado por otro participante, podrás editarlo cuando éste termine y desbloquee el documento.
  • Azul: El contrato está firmado. No se puede editar en este punto.
  • Gris: Estás viendo una “versión antigua” del documento. Nadie puede editar una versión antigua, solo la última versión es editable.
