Ayuda

Ver detalles de un contrato

Asegúrate de tener claros los requisitos de la API REST de Tractis antes de empezar la integración.

Descripción

Esta acción permite ver los detalles de un contrato concreto.

La acción se invoca con: GET /contracts/#{id}. Donde #{id} será el ID del contrato.

Ejemplo de invocación


  $ curl -u demo@tractis.com:password \
         -H 'Accept: application/xml' \
         -i -X GET https://www.tractis.com/contracts/497858506
