¿Como puedo utilizar Tractis LTA SOAP v1?

Para poder utilizar Tractis LTA, primero deberás crear una API Key de Tractis LTA y, a continuación, integrar dicha API Key en tu aplicación o servicio web.

Importante: Recomendamos usar la API REST v2.

Clientes LTA SOAP v1

Este servicio es ofrecido mediante una interficie de webservice implementando un protocolo propio de Tractis mediante SOAP. Puedes encontrar el archivo de definición del servicio en:

https://api.tractis.com/lta/content_storage.wsdl

Para invocar el servicio puedes utilizar alguno de los clientes que proporcionamos o desarrollar uno propio:

  • Cliente Tractis LTA para Java.
  • Cliente Tractis LTA para Ruby.

Instrucciones de instalación

Las instrucciones de instalación y uso están incluídas en los detalles de cada uno de los Clientes LTA.

URL de invocación

La URL de invocación del servicio es:

https://api.tractis.com/lta
