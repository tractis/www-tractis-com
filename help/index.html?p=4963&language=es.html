¿Cómo puedo crear una cuenta en Tractis utilizando mi certificado electrónico?

Para crear una cuenta en Tractis utilizando tu certificado electrónico sigue estos pasos:

  1. Visita la página de registro.
  2. Asegúrate de que está seleccionada la pestaña “Certificado“.
  3. Si vas a utlizar un certificado en tarjeta inteligente o dispositivo USB, introduce la tarjeta o conecta el USB.
  4. Rellena la información solicitada y pulsa el botón “Crea una cuenta en Tractis“.
  5. Tras crear tu cuenta, Tractis te enviará un email a la dirección de correo que hayas indicado para comprobar que es correcta. Pulsa el enlace que encontrarás en el email.
  6. Ya tienes una Cuenta en Tractis. A partir de ahora, puedes iniciar sesión en los servicios de Tractis utilizando tu certificado electrónico.

Este método de creación de cuenta está recomendado para aquellos usuarios que desean utilizar el mayor nivel de seguridad disponible a la hora de acceder a Tractis.
