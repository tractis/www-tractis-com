¿Como puedo obtener una API Key de Tractis Verifications?

Creación de la Verificación de Identidad

Las Pasarelas de Autenticación Fuerte de Tractis se realizan mediante nuestras Verificaciones de Identidad. Crear una Verificación de Identidad para una Pasarela de Autenticación es tan simple como:

  1. Iniciar sesión en Tractis y acceder a la dirección http://www.tractis.com/identity_verifications.
  2. Generar una API Key para el sitio web en el que quieras integrar la pasarela

Una vez creada la Verificación de Identidad no tienes más que integrar la Pasarela en tu sitio web.

Integración de la Verificación de Identidad

Actualmente dispones de dos métodos para integrar las verificaciones en tus servicios:

  1. Verificaciones de Identidad con Tractis: Solo tienes que poner un formulario HTML en tu web y hacer una sencilla validación de los datos que respondemos. Su uso es gratuíto, sin límite de autenticaciones.
  2. Módulo de Autenticación Tractis para Apache: tus usuarios nunca abandonan tu sitio web para autenticarse. Este método de integración requiere la instalación de un plug-in SSL/TLS 2 en tu servidor. Su uso tiene un coste de 0,05 € por autenticación, sin límite de autenticaciones.

Configuración opcional

De manera opcional puedes configurar la Verificación de Identidad:

  1. Especifica los atributos que deseas verificar utilizando los enlaces de “Activar/Desactivar” situados al lado de cada atributo.
  2. Especifica los tipos de certificados aceptados. Puedes aceptar todos los certificados capaces de verificar esos atributos o prohibir algunos. Date cuenta de que no todos los perfiles de certificados soportan todos los atributos y, por tanto, la activación de atributos (en el paso anterior) puede limitar el número de perfiles de certificados que podrás aceptar para realizar la verificación (p.ej: si activas el atributo “Fecha de nacimiento“, te desaparecerán de la lista de certificados todos aquellos que no contengan ese atributo).
  3. Especifica el método HTTP (GET/POST) con el cual se devolverá al usuario a tu sitio web una vez realizada la verificación de identidad
