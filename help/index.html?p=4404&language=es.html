Ayuda

¿Como puedo utilizar Tractis TSA?

Para poder utilizar Tractis TSA, primero deberás crear una API Key de Tractis TSA y, a continuación, integrar dicha API Key en tu aplicación o servicio web.

Clientes TSA

Tractis TSA puede generar sellos de tiempo tanto en formato XAdES 1.3.2 como en formato RFC 3161 (CMS).

Para la generación de sellos de tiempo en formato XAdES 1.3.2, debes invocar el servicio a través del protocolo de comunicación DSS, en su versión 1.0 y utilizando el perfil de Timestamp. Puedes utilizar alguno de los clientes que proporcionamos o desarrollar uno propio:

  • Cliente Tractis TSA XAdES 1.3.2. para Java.
  • Cliente Tractis TSA XAdES 1.3.2. para Ruby.

Para la generación de sellos de tiempo en formato RFC 3161 (CMS) debes seguir las indicaciones definidas en este RFC. Puedes utilizar alguno de los clientes que proporcionamos, desarrollar uno propio, o bien utilizar cualquier cliente de TSA RFC 3161 disponible en el mercado:

  • Cliente Tractis TSA RFC3191 para PHP.
  • Cliente Tractis TSA RFC3191 para Java (próximamente).

Instrucciones de instalación

Las instrucciones de instalación y uso están incluídas en los detalles de cada uno de los Clientes TSA.

URL de invocación

La URL de invocación del servicio para la generación de sellos de tiempo en formato XAdES 1.3.2. es:

https://api.tractis.com/tsa

La URL de invocación del servicio para la generación de sellos de tiempo en formato RFC3161 es:

https://api.tractis.com/rfc3161tsa
